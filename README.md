# Python 游戏集合

这是一个包含两个经典游戏的 Python 游戏集合，使用 Pygame 库开发。

## 游戏列表

### 1. 贪吃蛇游戏 (Snake Game)

一个增强版的贪吃蛇游戏，包含以下特性：
- 经典贪吃蛇玩法
- 墙体穿越功能
- 特殊食物系统
- 障碍物系统
- 游戏速度随分数增加
- 最高分记录功能
- 美观的游戏界面

### 2. 数字华容道 (Number Puzzle)

一个经典的数字华容道游戏，包含以下特性：
- 3x3 数字拼图
- 随机生成可解的初始状态
- 步数记录
- 胜利判定和庆祝界面
- 简洁直观的操作方式

## 安装和运行

### 前提条件

- Python 3.x
- Pygame 库

### 安装 Pygame

```bash
pip install pygame
```

### 运行游戏

1. **贪吃蛇游戏**：
   ```bash
   cd snake_game
   python enhanced_snake.py
   ```

2. **数字华容道**：
   ```bash
   cd number_puzzle
   python number_puzzle.py
   ```

## 游戏操作说明

### 贪吃蛇游戏
- **方向键**：控制蛇的移动方向
- **空格键**：开始游戏
- **R 键**：游戏结束后重新开始
- **Q 键**：退出游戏
- **M 键**：返回主菜单

### 数字华容道
- **鼠标点击**：点击与空格相邻的方块进行移动
- **空格键**：开始游戏
- **R 键**：重新开始游戏
- **Q 键**：退出游戏

## 游戏特性详解

### 贪吃蛇游戏特性
1. **墙体穿越**：蛇可以穿过屏幕边缘，从另一侧出现
2. **特殊食物**：定期出现的黄色特殊食物，提供更多分数和长度
3. **障碍物**：游戏开始时随机生成的紫色障碍物，增加游戏难度
4. **速度系统**：随着分数增加，游戏速度会逐渐提高
5. **最高分记录**：游戏会自动保存并显示最高分

### 数字华容道特性
1. **可解性保证**：游戏会确保生成的初始状态是可解的
2. **步数统计**：实时显示玩家移动的步数
3. **胜利检测**：自动检测是否完成拼图
4. **胜利界面**：完成拼图后显示庆祝界面和总步数
